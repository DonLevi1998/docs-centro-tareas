# Arquitectura

La solución está compuesta por dos grandes módulos:

- **Backend**: API REST desarrollada en .NET, responsable de la lógica de negocio, persistencia y seguridad.
- **Frontend**: Aplicación Angular para la gestión visual de tableros y tareas.
